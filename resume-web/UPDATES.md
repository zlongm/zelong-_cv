### 色系自定义 (进阶)

如果想改变颜色，编辑 `styles.css`:

```css
:root {
    --primary-color: #FF6B35;      /* 改这个 */
    --secondary-color: #FFA500;    /* 改这个 */
    --accent-color: #FFB84D;       /* 改这个 */
}
```

可选的暖色方案:
- **日落红**: #FF6B35, #FF8C42, #FFB84D
- **秋天金**: #D2691E, #FF8C00, #FFD700
- **温暖橙**: #FF7F50, #FF9900, #FFD580
- **深棕金**: #8B4513, #CD853F, #DAA520
