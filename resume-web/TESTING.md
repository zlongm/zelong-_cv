## ⚡ 性能检查

```
1. 按 F12 打开开发者工具
2. 选择 "Performance" 标签
3. 点击 "Record" 并刷新页面
4. 停止 Recording
5. 查看:
   ✓ 首屏加载时间 < 2秒
   ✓ Main Thread 使用率 < 50%
```

## 🔍 常见问题排查

### 问题: 动画不流畅
```
解决:
- 清除浏览器缓存 (Ctrl+Shift+Delete)
- 关闭浏览器扩展
- 尝试其他浏览器
```

### 问题: PDF看起来不好
```
解决:
- 确保有 html2pdf.js 库
- 尝试不同的打印设置
- 在 styles.css @media print 中调整
```

### 问题: 颜色看起来不对
```
解决:
- 检查 styles.css 中的 :root 变量
- 确保浏览器支持 CSS 变量
- 清除缓存后重新加载
```
